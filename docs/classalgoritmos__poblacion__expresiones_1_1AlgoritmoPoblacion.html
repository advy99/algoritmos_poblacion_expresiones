template<class T>
class algoritmos_poblacion_expresiones::AlgoritmoPoblacion< T >

Clase AlgoritmoPoblacion.

Una instancia del tipo AlgoritmoPoblacion representará un estimador para los datos dados, utilizando una poblacion del tipo dado.

Parámetros del template
TTipo de la población que utilizará el algoritmo
