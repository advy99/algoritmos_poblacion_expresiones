Parámetros del template
Ttipo de los datos a separar
Parámetros

etiquetasEtiquetas de los datos a separar
PORCENTAJE_TESTPorcentaje de datos que formaran parte del conjunto de test, por defecto 0.2
COMIENZOIndice a partir de donde los datos pasaran a ser de test. Si vale -1, se escogeran los datos de test de forma aleatoria. Por defecto -1
